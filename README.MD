# 🛡 CryptoShield AI — Telegram Bot

Бот принимает скам-сообщения, извлекает крипто-адреса через AI, сохраняет в базу.

---

## Быстрый старт (15 минут)

### Шаг 1 — Получи токен бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя: `CryptoShield AI`
4. Придумай username: `cryptoshield_ai_bot` (или любой свободный)
5. Скопируй токен вида `7123456789:AAF...`

### Шаг 2 — Получи Anthropic API ключ

1. Зайди на https://console.anthropic.com
2. Зарегистрируйся
3. Перейди в API Keys → Create Key
4. Скопируй ключ вида `sk-ant-api03-...`

> 💡 На новом аккаунте есть $5 бесплатных кредитов — хватит на ~500 анализов

### Шаг 3 — Деплой на Railway (бесплатно)

1. Создай аккаунт на https://railway.app (через GitHub)
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи этот проект в GitHub (или нажми **Deploy from template**)
4. Перейди в **Variables** и добавь:
   ```
   TELEGRAM_BOT_TOKEN = твой_токен
   ANTHROPIC_API_KEY  = твой_ключ
   ```
5. Railway автоматически запустит бот через `Procfile`

### Альтернатива — запуск локально

```bash
# Клонируй или скопируй файлы в папку
cd cryptoshield-bot

# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Открой .env и вставь свои ключи

# Запусти
python bot.py
```

## Что умеет бот

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие и инструкция |
| `/help` | Помощь |
| `/check 0x123...` | Проверить конкретный адрес |
| `/stats` | Статистика базы |
| `/top` | Топ-10 адресов с жалобами |
| _(любой текст)_ | Анализирует как скам-сообщение |

Пользователь просто пересылает скам-сообщение боту — бот делает всё сам.

## База данных

SQLite файл `scam_reports.db` создаётся автоматически. Таблицы:

- **reports** — все входящие репорты с текстом
- **addresses** — уникальные адреса с risk score
- **report_addresses** — связь репортов с адресами
- **confirmations** — подтверждения от других пользователей

### Экспорт данных

```python
from database import Database
db = Database("scam_reports.db")
db.export_csv("addresses_export.csv")
```

## Risk Score — как считается

| Score | Значение |
|-------|---------|
| 0–39 | 🟢 Не найден / низкий риск |
| 40–69 | 🟡 Подозрительный |
| 70–95 | 🔴 Высокий риск — скам |

Начальное значение: **55** (одна жалоба)  
+5 за каждую новую жалобу  
+3 за каждое подтверждение от другого пользователя  
Максимум: **95** (оставляем место для ручной верификации)
